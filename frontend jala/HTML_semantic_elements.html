Good to Know

Difference between <span> and <div>

  • <span>: An inline element used to group inline elements for styling purposes.
  • <div>: A block-level element used to group block-level elements and structure the HTML document.

Difference between HTML4 and HTML5

  • <header>: HTML4 uses <div> elements with classes or IDs for headers, while HTML5 introduces the semantic <header> element.
  • <menu>: In HTML4, <menu> is less commonly used, whereas in HTML5, it is a semantic element for a list of commands or a toolbar.
  • <content>: HTML4 does not have a specific <content> element; HTML5 introduces elements like <section> and <article> for content.
  • <article>: HTML4 uses <div> for articles, while HTML5 introduces the semantic <article> element to represent independent content.
  • <footer>: In HTML4, footers are usually implemented using <div> elements, while HTML5 provides a semantic <footer> element.
