Things to Know

  • Difference between visibility:hidden and display:none?
    `visibility: hidden;` makes the element invisible, but it still occupies space in the layout. `display: none;` completely removes the element from the document flow, so it doesn’t occupy any space.
  • Difference between inline and block level elements?
    Inline elements only take up as much space as necessary and flow with the text (e.g., <span>, <a>). Block-level elements take up the full width available, starting on a new line (e.g., <div>, <p>).
  • What are selectors?
    Selectors are patterns used in CSS to select the elements you want to style. Examples include type selectors (e.g., p), class selectors (e.g., .class), ID selectors (e.g., #id), and attribute selectors.
  • Which property is used to set the line style for the outline?
    The `outline-style` property is used to set the line style for the outline. It can be set to values like `solid`, `dashed`, `dotted`, etc.
